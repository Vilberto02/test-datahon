# Carpeta de Área de Trabajo (Workspace)

Esta carpeta es el área de trabajo durante la participación en la datathon. Aquí se desarrolló el código fuente principal, aplicando los conocimientos adquiridos en la fase de entrenamiento.

## Archivos

- `scripts`: Contiene scripts de Python para el entrenamiento del modelo de predicción.
- `models`: Modelos de machine learning entrenados y utilizados para hacer predicciones.
- `notebooks`: Notebooks de Jupyter utilizados para análisis exploratorio, limpieza de datos, y visualizaciones.
